Featured Project — Firefighter Robot Car (IoT)

Firefighter Robot Car
Arduino UNO • Sensors • Autonomous navigation

Designed and implemented a firefighter robot car using Arduino UNO, temperature & flame sensors, motor drivers and custom control logic. The robot detects fire sources, navigates small indoor obstacle courses, and activates extinguishing mechanism. I handled hardware integration, sensor calibration, and end-to-end firmware coding (C/C++). This project demonstrates embedded systems design, real-time control and IoT telemetry.

  • Microcontroller: Arduino UNO (firmware in Arduino C)
  • Peripherals: Flame sensor, DHT/temperature, ultrasonic sensor, L298N motor driver
  • Features: Autonomous fire-detection, basic path planning, remote telemetric logs
